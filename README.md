BSB64
=====================

BSB64 is a specification and implementation for the simple encryption.

## Usage
Java:
```Java
BSB64.encodeString("abc", 1);
BSB64.decodeString("wsTG", 1);
```

JavaScript:
```JavaSctipt
BSB64.encodeString('abc', 1);
BSB64.decodeString('wsTG', 1);
```

Visual Basic:
```Visual Basic
BSB64.encodeBSB64Str("abc", 1)
BSB64.decodeBSB64Str("wsTG", 1)
```
